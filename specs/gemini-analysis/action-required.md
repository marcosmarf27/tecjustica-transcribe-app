# Ações Manuais: Análise de Transcrições com Gemini API

Passos que precisam ser executados manualmente por um humano.

## Antes da Implementação

- [ ] **Obter Gemini API Key** - Acessar [aistudio.google.com/apikey](https://aistudio.google.com/apikey) e criar uma chave gratuita para testar a integração durante o desenvolvimento

## Após a Implementação

- [ ] **Configurar API Key no app** - Abrir Configurações e inserir a Gemini API Key para testar o fluxo completo
- [ ] **Testar com transcrição real** - Executar uma transcrição completa e usar o botão "Analisar com IA" para verificar que os tópicos retornados fazem sentido e os timestamps correspondem ao áudio/vídeo
